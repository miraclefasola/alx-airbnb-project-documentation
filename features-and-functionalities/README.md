## Key Features Overview

## 1. User Authentication & Authorization

Secure user registration and login with email/password.

Role-based access (guest, host, admin).

JWT-based authentication.

Password hashing and recovery support.

## 2. Property Management

Hosts can:

Create, update, and delete property listings.

Upload images and add detailed descriptions.

Guests can:

Browse and filter available properties by price, location, and amenities.

Admins can:

Verify or remove properties violating policies.

## 3. Search & Filtering

Filter properties by:

Location

Price range

Availability

Amenities

Sort by:

Rating

Price

Popularity

Pagination for large result sets.

## 4. Booking Management

Create Bookings:

Guests can book properties for specific dates.

Prevent double bookings via date validation.

Cancel Bookings:

Guests or hosts can cancel under certain policies.

Track Status:

pending, confirmed, canceled, or completed.

## 5. Payment Integration

Integration with secure gateways (e.g., Stripe, PayPal, Paystack, Flutterwave).

Handle:

Upfront payments from guests.

Automatic payouts to hosts.

Support for multiple currencies.

## 6. Reviews & Ratings

Guests can leave property reviews after completed bookings.

Hosts can respond to reviews.

Each review is tied to a valid booking to prevent fake reviews.

## 7. Notification System

Email and in-app notifications for:

Booking confirmations

Cancellations

Payment updates

Admin alerts for disputes or failed transactions.

## 8. Admin Dashboard

Manage all system entities:

Users

Properties

Bookings

Payments

Monitor reports, analytics, and disputes.

Approve or deactivate listings or users.
